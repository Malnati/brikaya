# Referências Técnicas

Este diretório contém bibliotecas e documentação de referência utilizadas no projeto BrickBreaker.

## Conteúdo disponível

### Heroicons
Biblioteca de ícones otimizados para React e desenvolvimento web.

- **Localização:** `./heroicons/`
- **Versão:** Conjunto otimizado de ícones 24px solid
- **Uso:** Ícones para interface do jogo e componentes UI
- **Formatos:** SVG otimizados, React components, Vue components

### Swagger
Documentação de referência para APIs REST.

- **Localização:** `./swagger/`
- **Conteúdo:** Exemplos de especificação OpenAPI e templates
- **Uso:** Referência para documentação de APIs futuras

## Como utilizar

Os ícones Heroicons podem ser importados diretamente nos componentes React:

```tsx
import { PlayIcon, PauseIcon } from '@heroicons/react/24/solid';
```

Para mais informações, consulte a documentação específica de cada biblioteca.
