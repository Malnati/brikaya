# Autoria e propriedade

Brikaya é uma obra proprietária de Ricardo Malnati.

## Diretrizes

- Manter autoria clara nos documentos públicos do repositório.
- Registrar origem de qualquer asset externo antes de incorporar ao runtime.
- Não incluir material de terceiros sem autorização compatível com distribuição web, PWA e monetização futura.
- Manter o projeto sem licença aberta até decisão expressa em contrário.

## Evidência interna

O histórico público passa a partir do baseline atual. O backup privado pré-consolidação fica fora do Git e serve apenas para recuperação pelo titular.
