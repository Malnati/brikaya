# Brikaya v1.2.0 — Modo Torreta

## Destaque

Esta versão adiciona o **Modo Torreta**, um modo alternativo acionado pelo menu. O jogador mira pela bolha de uma torreta esférica dentro do círculo, preservando a bolinha, os blocos, poderes, fases e recordes do Brikaya.

## Mudanças de produto

- Novo seletor `Modo de jogo` no menu.
- Nova opção `Torreta` ao lado de `Clássico`.
- Preferência salva para manter o modo escolhido nas próximas partidas.
- Reinício automático da partida ao trocar de modo.
- Visual com profundidade, mira, cano, vidro e vinheta esférica.

## Qualidade e cobertura

- Testes unitários para modo, preferência, passagem de props, renderização e motor do jogo.
- Teste de aplicação para troca pelo menu e persistência.
- QA visual automatizada desktop/mobile para `?qaScenario=ball-turret`.
- Evidências em `docs/assets/issues/ball-turret-mode/evidence/`.
