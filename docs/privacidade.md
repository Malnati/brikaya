# Privacidade

Brikaya armazena preferências, pontuações e estatísticas no navegador do jogador.

## Dados locais

- Pontuação.
- Preferências de aparência e áudio.
- Estado técnico necessário para funcionamento offline.

## Diretrizes

- Não coletar dados pessoais no jogo principal sem nova aprovação.
- Não adicionar rastreamento externo ao runtime sem revisão de privacidade.
- Manter anúncios separados da experiência principal: somente em algumas transições entre fases, a cada 3 fases concluídas, nunca durante uma jogada ativa e nunca como requisito para jogar.
- Usar uma plataforma de consentimento certificada pelo Google antes de anúncios personalizados no Espaço Econômico Europeu, Reino Unido e Suíça.
- Continuar a partida sem bloqueio quando anúncio, consentimento, internet ou demanda não estiverem disponíveis.
