## Destaque

- Introdução do jogo agora exibe textos letra-por-letra (efeito typing) com duração total de 5 segundos
- Possibilidade de pular a animação clicando na tela

## Mudanças

- Novo hook `useTypingEffect` para efeito de digitação progressiva
- Duração da animação de introdução aumentada de 3.5s para 5s
- Textos aparecem sequencialmente: BRIKAYA (0-800ms), título (900-1900ms), descrição (2000-4000ms)
- Cursor pointer no overlay indica que é clicável para skip
- Timeout dos testes E2E ajustado de 5.5s para 8s para acomodar nova duração
