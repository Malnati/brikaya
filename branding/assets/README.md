# Brikaya Brand Assets Directory

Este diretório contém os ativos visuais e tokens de identidade para o jogo Brikaya.

## Logos e ícones

- `app-logo.svg`: logotipo vetorial do jogo (quando disponível).
- Logos de terceiros (cliin-logo-*.png): mantidos para referência histórica, podem ser substituídos por assets específicos do jogo.

## Tokens de identidade

- **Nome do jogo:** Brikaya
- **Paleta principal:** 
  - Primária: `#1a1a1a` (fundo escuro)
  - Secundária: `#2d2d2d` (container)
  - Destaque: `#00d4ff` (ciano brilhante - elementos do jogo)
- **Estilo:** Arcade retro-moderno com tema escuro

## Diretrizes de uso

- ✅ Manter contraste adequado para legibilidade (WCAG AA mínimo)
- ✅ Usar cores do jogo consistentemente em todos os elementos
- ✅ Preservar identidade visual arcade/retro
- ❌ Não usar fontes externas (Google Fonts) - apenas system fonts
