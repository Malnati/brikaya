Excluir dados do jogo neste aparelho

  • Abra as configurações do navegador.
  • Procure dados do site ou armazenamento do site.
  • Encontre brikaya.com.
  • Escolha limpar ou remover dados do site.
  • Abra Brikaya novamente para começar com dados locais novos.

O que pode ser apagado

Pontuação, recordes, preferências, consentimentos, idioma escolhido e histórico local podem ser removidos ao limpar os dados do site no navegador.

Conta de jogador

Brikaya não cria conta de jogador nesta versão. Portanto, não há perfil de jogador hospedado pelo Brikaya para excluir.

Solicitações por e-mail

Se você enviou contato para suporte ou privacidade e quer solicitar remoção de uma mensagem ou dado associado ao atendimento, escreva para contato@brikaya.com com o assunto “Exclusão de dados”.

Responderemos usando o mesmo canal sempre que possível. Podemos manter registros mínimos quando necessário para segurança, prevenção de abuso, cumprimento legal ou defesa de direitos.
