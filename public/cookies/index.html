Cookies e anúncios

Escolhas salvas

Brikaya pode salvar escolhas no navegador para manter idioma, som, aparência, consentimentos e progresso entre partidas.

Anúncios

Quando anúncios estiverem disponíveis, fornecedores de anúncios podem usar cookies, armazenamento do navegador ou identificadores semelhantes para veicular, limitar, medir e proteger anúncios.

Anúncios não são necessários para jogar e não aparecem durante uma jogada ativa.

Consentimento

Em regiões onde a lei exige escolha prévia, Brikaya deve pedir consentimento antes de usar dados para personalização de anúncios. Se você negar ou se o consentimento não estiver disponível, o jogo continuará funcionando sem bloquear a próxima fase.

Como controlar

  • Use o menu do jogo para revisar consentimentos disponíveis.
  • Use as configurações do navegador para apagar dados de brikaya.com.
  • Use controles do navegador ou do aparelho para limitar cookies e anúncios personalizados.
